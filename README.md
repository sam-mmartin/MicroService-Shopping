# Geek Shopping - Microservices

#### Dotnet CLI

```bash

# Adiciona um pacote ao projeto
$ dotnet add package nomeDoPacote

# Adiciona um pacote a um projeto de uma solução
$ dotnet add pastaDoProjeto/nomeDoProjeto package nomeDoPacote

# Install global Aspnet Codegenerator
$ dotnet tool install -g dotnet-aspnet-codegenerator

# Running the project
$ dotnet run

# Generate Razor Views
$ dotnet-aspnet-codegenerator view Viewname Template -m Model --useDefaultLayout -outDir Views/ControllerDir

```
